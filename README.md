## 📌 About

Orbit is a platform built during the **University of Sheffield SWiCS & FinTech Hackathon** to help students connect intelligently using:

- Academic field alignment  
- Shared experiences  
- Compatibility signals  
- Collaboration potential  

Instead of relying on chance connections, Orbit aims to make academic networking faster, more meaningful, and more actionable.

---

## ✨ Features

- **Student discovery** based on academic profile and interests  
- **Compatibility signals** to prioritize stronger matches  
- **Collaboration potential** indicators for project teaming  
- **AI concept exploration** for:
  - match explanations   
  - communication dynamics  

---

## 🧠 How It Works

1. Users create a student profile (field, interests, experience, goals)  
2. Data is stored securely using Firebase  
3. Matching logic scores compatibility using defined signals  
4. The platform surfaces ranked matches and supports future AI extensions

> **AI Extension (Concept):** An LLM layer can explain *why* two students match, recommend group pairings, and suggest communication approaches.

- **Backend / BaaS:** Firebase
